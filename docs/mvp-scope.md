# MVP Scope Freeze

## Must Have
- Auth with roles: `customer`, `kitchen`, `owner`.
- Menu listing and menu management.
- Cart and checkout flow.
- Order lifecycle and status history.
- Kitchen board realtime updates.
- Admin basic dashboard.

## Should Have
- Wait-time highlighting in kitchen board.
- Basic reports for daily totals.

## Out of Scope
- Multi-restaurant tenancy.
- Inventory forecasting.
- Advanced campaign management.

## Constraints
- Solo development.
- 10-14 week timeline.
- Use managed services where possible.
