# User Stories (MVP)

## Customer
- As a diner, I can scan a QR to start ordering for my table.
- As a diner, I can browse menu items and customize selections.
- As a diner, I can place order and pay at counter.
- As a diner, I can track order progress in real time.

## Kitchen
- As a cook, I can see all active orders grouped by status.
- As a cook, I can drag/drop orders between pending, preparing, and ready.
- As a cook, I can quickly update status with one click.

## Owner/Admin
- As an owner, I can create/update/disable menu items.
- As an owner, I can view live order queue.
- As an owner, I can view basic daily sales and order count.

## Acceptance Criteria
- Status changes reflect on all screens within 2 seconds.
- Only authenticated staff can access kitchen and admin routes.
- Order history persists with status timeline for audit.
